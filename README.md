# 🚀 Personal Portfolio Landing Page

> Uma interface de portfólio elegante e bem estruturada, projetada para destacar a identidade visual e as competências de um profissional de forma clara e moderna.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir uma **Landing Page de Portfólio** que combine uma estética minimalista com funcionalidades essenciais de navegação e conversão (CTAs).

O foco do desenvolvimento foi criar uma estrutura semântica que suporte uma identidade visual forte, permitindo que informações sobre carreira, serviços e redes sociais sejam acessadas de forma fluida.

## ✨ Destaques e Diferenciais Técnicos

* **Arquitetura de Navegação Completa:** Implementação de uma barra de navegação (`navbar`) que inclui links estruturados e um componente de busca integrado, demonstrando atenção à usabilidade.
* **Seção Hero Dinâmica:** Uso de containers específicos para textos de apresentação e profissões dinâmicas (`typing-text`), criando uma introdução impactante para o usuário.
* **Integração de Iconografia (Font Awesome):** Utilização da biblioteca Font Awesome (v5.15.2) para ícones de busca e redes sociais, garantindo uma linguagem visual profissional e escalável.
* **Hierarquia Visual e Branding:** Organização clara de logotipos, imagens de destaque e botões de chamada para ação (CTA), como "Sobre mim" e "Siga-me".
* **Gestão de Mídias Sociais:** Container dedicado (`media-icons`) para links externos de plataformas como Facebook, Twitter, LinkedIn e Pinterest, centralizando a presença digital.
* **Organização Profissional de Código:** Separação rigorosa entre a marcação estrutural (`index.html`) e a identidade visual localizada no diretório `css/style.css`.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e organização de seções.
* **CSS3:** Gerenciamento de layout, tipografia e design responsivo.
* **Font Awesome:** Biblioteca de ícones vetoriais para interface e social media.
